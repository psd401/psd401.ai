## Summary

AI Studio is an open-source platform developed by Peninsula School District that provides secure, cost-effective access to frontier generative AI models for K-12 educators and students. Unlike individual subscriptions, AI Studio uses a district-hosted API architecture that reduces costs by 90% while maintaining data privacy and security. The platform enables users to switch between multiple cutting-edge models and create custom AI assistants without coding.

## Features

**Frontier AI Model Access**

- Switch between OpenAI GPT-5, Anthropic Claude Opus, Google Gemini, and more
- Real-time model comparison for different tasks
- Access to the same models used in universities and businesses

**Custom AI Assistants**

- No-code assistant creation using prompt chains
- Parallel prompt execution for complex tasks
- Custom chat models for instructional and operational needs

**District-Level Security**

- Data processed within district's secure servers
- No student or staff identifying information leaves PSD environment
- Secure API calls managed internally

**Cost-Effective Architecture**

- 90% cost reduction compared to individual licenses
- Centralized API management
- No per-user subscription fees

**Open Source**

- MIT licensed and available on GitHub
- Self-hosting capabilities
- Active development and community contributions

## Use Cases

**For Educators**

- Generate lesson plans across multiple AI models
- Create custom instructional assistants for different subjects
- Compare AI responses for research and curriculum development
- Automate administrative tasks with tailored AI tools

**For Students**

- Access frontier AI models for homework and research
- Practice critical thinking by comparing different AI responses

**For Schools & Districts**

- Deploy secure AI platform without third-party data sharing

- Self-host or use hosted solutions
